# Changelog

This file records user-visible changes. Versions are listed in reverse
chronological order.

## Unreleased

## 0.1.8 - 2026-07-17

### Added

- `aicommit uninstall` removes a direct binary installation; add `--purge` to remove saved configuration too.
- `aicommit -v` and `aicommit --version` provide shortcuts for version information.

## 0.1.7 - 2026-07-17

### Breaking Changes

- Reworked configuration commands: `aicommit ai` and `aicommit ai --setup` have been removed. Use `aicommit config` and `aicommit config setup` instead.

## 0.1.6 - 2026-05-14

### Added

- Configurable commit-message language.
